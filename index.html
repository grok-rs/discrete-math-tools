    Algebra of Sets
    Identity Laws
    A ∪ ∅ = A
    A ∩ U = A
    Domination Laws
    A ∪ U = U
    A ∩ ∅ = ∅
    Idempotent Laws
    A ∪ A = A
    A ∩ A = A
    Complement Laws
    A ∪ A' = U
    A ∩ A' = ∅
    (A')' = A
    U' = ∅
    ∅' = U
    Commutative Laws
    A ∪ B = B ∪ A
    A ∩ B = B ∩ A
    Associative Laws
    A ∪ (B ∪ C) = (A ∪ B) ∪ C
    A ∩ (B ∩ C) = (A ∩ B) ∩ C
    Distributive Laws
    A ∪ (B ∩ C) = (A ∪ B) ∩ (A ∪ C)
    A ∩ (B ∪ C) = (A ∩ B) ∪ (A ∩ C)
    De Morgan's Laws
    (A ∪ B)' = A' ∩ B'
    (A ∩ B)' = A' ∪ B'
    Absorption Laws
    A ∪ (A ∩ B) = A
    A ∩ (A ∪ B) = A
    Difference
    A \ B = A ∩ B'
    A \ (B ∪ C) = (A \ B) ∩ (A \ C)
    A \ (B ∩ C) = (A \ B) ∪ (A \ C)
    Symmetric Difference
    A △ B = (A \ B) ∪ (B \ A)
    A △ B = (A ∪ B) \ (A ∩ B)
    A △ B = B △ A
    A △ ∅ = A
    A △ A = ∅
